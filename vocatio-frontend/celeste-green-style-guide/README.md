# Celeste Green Style Guide

Welcome to the Celeste Green Style Guide! This document outlines the design principles and guidelines for our project, focusing on a consistent aesthetic using celeste green and neutral colors. 

## Color Palette

- **Celeste Green**: #A8E6CE
- **Dark Celeste Green**: #5D9B77
- **Light Celeste Green**: #D1F2E7
- **Neutral Light**: #F5F5F5
- **Neutral Medium**: #B0B0B0
- **Neutral Dark**: #4F4F4F
- **White**: #FFFFFF
- **Black**: #000000

## Typography

- **Primary Font**: 'Roboto', sans-serif
- **Headings**: 
  - H1: 36px, Bold, Dark Celeste Green
  - H2: 30px, Bold, Neutral Dark
  - H3: 24px, Semi-Bold, Neutral Medium
- **Body Text**: 
  - Size: 16px, Regular, Neutral Dark
  - Line Height: 1.5
- **Small Text**: 
  - Size: 14px, Regular, Neutral Medium

## Spacing

- **Base Unit**: 8px
- **Small Spacing**: 4px
- **Medium Spacing**: 16px
- **Large Spacing**: 32px
- **Extra Large Spacing**: 64px

## Button Styles

- **Primary Button**:
  - Background Color: Celeste Green
  - Text Color: White
  - Padding: 12px 24px
  - Border Radius: 4px
  - Font Size: 16px
  - Hover Effect: Dark Celeste Green background
- **Secondary Button**:
  - Background Color: Neutral Light
  - Text Color: Neutral Dark
  - Padding: 12px 24px
  - Border Radius: 4px
  - Font Size: 16px
  - Hover Effect: Neutral Medium background

## Form Elements

- **Input Fields**:
  - Border: 1px solid Neutral Medium
  - Border Radius: 4px
  - Padding: 10px
  - Font Size: 16px
  - Focus State: Border color changes to Celeste Green
- **Text Area**:
  - Same styles as input fields, with increased height for better usability.

## Additional Guidelines

- **Icons**: Use simple line icons in Neutral Medium for a clean look.
- **Images**: Ensure images are high quality and maintain a consistent aspect ratio.
- **Accessibility**: Ensure all text has sufficient contrast against backgrounds for readability.

By adhering to this style guide, we can maintain a cohesive and visually appealing design throughout our project.
